# 1. Alertmine

- **Concepto:** AlertMine es una aplicación de alertas para entornos mineros. Permite enviar notificaciones visuales a los empleados a través de una aplicación móvil, que se administra desde una interfaz web.
- **Arquitectura:**
    - Aplicación móvil (Android, teléfono en el showroom): Recibe las alertas.
    - Aplicación web: Permite administrar las alertas y los usuarios.
    - Base de datos: Almacena la información (usuarios, alertas, etc.).
- **Recursos:**

# 2. Gafas RA/RV Meta Quest 3

**Puntos clave:**
- **Integración RAFST:** La integración directa de MQ3 con Remote Assist, Field Service y Teams es limitada. Microsoft Mesh es una alternativa para reuniones en RV.
- **Valor de RA/RV:** Formación mejorada, reducción de costes (menos desplazamientos, prototipos virtuales) y demostraciones innovadoras a clientes.
- **Desarrollo Unity:** Usar Unity LTS, Open XR, XR Interaction Toolkit. Considerar SideQuest como alternativa a Meta Quest Link si no funciona.

**Acciones recomendadas:**
1. **Explorar Microsoft Mesh:** Evaluar su uso para reuniones y colaboración remota.
2. **Analizar casos de uso RA/RV:** Identificar áreas de formación, soporte y demostración donde la RA/RV pueda generar valor.
3. **Desarrollo Unity:** Iniciar proyectos de prueba con Unity, Open XR y el XR Interaction Toolkit.
4. **Investigar Mimbus:** Evaluar la posibilidad de adquirir licencias para uso profesional de RA/RV.

# 3. Maquetas

1. **Conexión con el Digital Twin:**
    - **Problema:** Conexión inexistente con el Digital Twin.
    - **Acción:** Diagnosticar la causa raíz de la desconexión, creo que es el router TP-Link
2. **Reparación Mecánica y Funcional:**
    - **Problema:** Mal funcionamiento del primer levier (palanca) y del lector NFC.
    - **Acción:** Inspeccionar el levier para identificar daños físicos o desgaste. Reparar o reemplazar según sea necesario. Verificar la conexión y configuración del lector NFC y de las piezas rojas/azules/blancas
3. **Adquisición de Compresor para Maquetas de Ciberseguridad:**
    - **Problema:** Falta de un compresor adecuado para el funcionamiento de las maquetas de ciberseguridad.
    - **Acción:** Determinar las especificaciones técnicas necesarias para el compresor (presión, flujo de aire, etc.). Investigar diferentes modelos y proveedores.
